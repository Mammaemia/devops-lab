# 02 — Nginx

Настройка веб-сервера Nginx с нуля.

## Что сделано

- Установлен и запущен Nginx
- Создан виртуальный хост на порту 8081
- Открыты порты в UFW (80, 8081)

## Структура Nginx

- `sites-available/` — все конфиги сайтов
- `sites-enabled/` — активные сайты (симлинки)

## Конфиги

- `configs/devops-lab` — конфиг виртуального хоста
